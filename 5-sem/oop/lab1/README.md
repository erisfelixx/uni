# Lab 1 - Railway Management System

Консольний додаток для управління рухомим складом пасажирського потяга. Реалізовано в рамках Лабораторної роботи №1 з дисципліни "Об'єктно-орієнтоване програмування".

## Варіант завдання (№8)

Транспорт. Визначити ієрархію рухомого складу залізничного транспорту. Створити пасажирський поїзд. Порахувати загальну чисельність пасажирів та багажу. Провести сортування вагонів на основі рівня комфортності. Знайти вагони у поїзді, що відповідають заданому діапазону параметрів числа пасажирів.

---

## Технології

- **Java 21**
- **Maven** (Build automation)
- **JUnit 5** (Unit Testing)
- **GSON** (JSON Serialization/Deserialization)

## Функціонал

**1. Ієрархія класів:** Реалізовано абстрактний клас `RollingStock` та похідні типи вагонів (`Luxury`, `Coupe`, `Placzkart`).

**2. Управління поїздом:** Створення поїзда, додавання вагонів (вручну через консоль або автоматично).

**3. Розрахунки:** Підрахунок загальної кількості пасажирів та ваги багажу.

**4. Сортування:** Сортування вагонів за рівнем комфорту (High -> Low).

**5. Пошук:** Фільтрація вагонів за заданим діапазоном кількості пасажирів.

**6. Збереження даних:** Запис та зчитування стану поїзда у файл `train.json` (із збереженням типів вагонів).

## Структура проєкту

```
src/main/java/org/example
├── model/      # Сутності (Вагони, Enum комфорту)
├── service/    # Бізнес-логіка (TrainManager)
├── utils/      # Робота з файлами (FileManager, GSON adapters)
└── Main.java   # Точка входу (Консольне меню)
```
