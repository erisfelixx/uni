# Lab 02 - Обробка XML документів засобами Java


## Предметна область (варіант 13)

**Тема:** Туристичні путівки. Туристичні путівки, що пропонуються агентством, мають такі характеристики: 
 - *Type* – тип путівки (вихідного дня, екскурсія, відпочинок, паломництво тощо). 
 - *Country* – країна для подорожі. 
 - *Number days/nights* – кількість днів та ночей. 
 - *Транспорт* – вид перевезення туристів (авіа, залізничні, авто, лайнер). 
 - *Hotel characteristic* (має бути кілька) – кількість зірок, чи включено харчування та яке (HB, BB, Al), який номер (1,2,3-х місні), чи є телевізор, кондиціонер тощо. 


---

## Функціонал

**1. XSD Схема:** Розроблено схему `vouchers.xsd` з використанням простих та комплексних типів, перерахувань (Enum), шаблонів (Regex) та обмежень (зірки 1-5).

**2. Java Модель:** Створено клас `TouristVoucher`, що повністю відповідає структурі XML.

**3. Парсинг:** Реалізовано 3 види парсерів через спільний інтерфейс VoucherParser:

 - **DOM** (Document Object Model)
 - **SAX** (Simple API for XML)
 - **StAX** (Streaming API for XML)

**4. Валідація:** Перевірка XML-файлу на відповідність схемі XSD перед обробкою.

**5. Сортування:** Реалізовано компаратори для сортування путівок (наприклад, за ціною).

**6. Трансформація (XSLT):** Автоматична генерація HTML-таблиці (vouchers_report.html) на основі XML даних.

**7. Unit-тести:** Повне покриття логіки парсерів тестами JUnit 5.

## Структура проєкту
 
```
src

│   ├── java/ua/knu/tourist

│   │   └── Main.java   
│   └── resources

    ├── java/ua/knu/tourist
    │   └── AppTest.java   
    └── resources
        └── test_vouchers.xml # тестові дані
```
